# Nasty Convert

> A Video coverter built with Electron, React, and Redux. Webpack server is implemented for hot reloading.

## Install

```bash
npm install
```

## Usage

<sub>\*Run the webpack server</sub>

```bash
npm start
```

<sub>\*Run the electron app</sub>

```bash
npm run electron
```
